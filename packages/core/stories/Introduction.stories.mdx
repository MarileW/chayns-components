## Overview

Welcome to the Storybook documentation for the **chayns-components**.
This library has been completely reworked using modern React patterns.
Use the navigation on the left to explore the available components.
Core components are listed under the `Core` section, while additional packages are grouped separately.

> <sup>1</sup> In the latest version, all components have been rewritten as functional components with React Hooks. Animations now use **Framer Motion**, and the component props have been simplified and improved for better clarity and usability.

## Installation

This library is modular. Core and additional packages must be installed separately using either `npm` or `yarn`.

<details>
    <summary>Install using NPM</summary>

    ```bash
    # Core components
    npm install @chayns-components/core

    # Additional components
    npm install @chayns-components/code-highlighter
    npm install @chayns-components/color-picker
    npm install @chayns-components/date
    ```

## Usage

First, make sure to install the core package and any additional components you need. Then, wrap your application using the `ChaynsProvider` and apply the color scheme via `ColorSchemeProvider`:

```tsx
import React from 'react';
import { ChaynsProvider } from 'chayns-api';

const AppWrapper = () => (
    <ChaynsProvider>
        <App />
    </ChaynsProvider>
);

export default AppWrapper;
```

```tsx
import React from 'react';
import { ColorSchemeProvider } from '@chayns-components/core';
import { getSite } from 'chayns-api';

const App = () => {
    const { color, colorMode } = getSite();

    return (
        <ColorSchemeProvider color={color} colorMode={colorMode}>
            <YourComponent />
        </ColorSchemeProvider>
    );
};

export default App;
```

## Contribution

Make sure you're using **Node.js v16** or higher. To get started:

```bash
# NPM
npm install

# Yarn
yarn install
```

Build the packages locally:

```bash
# NPM
npm run lerna:build

# Yarn
yarn run lerna:build
```

Start the local Storybook instance to begin development:

```bash
# NPM
npm run storybook

# Yarn
yarn run storybook
```

> **Note:** You can access the local Storybook at [http://localhost:6006](http://localhost:6006)
